# Implement Setup Pattern

## Description
Implement the canonical install/update flow for the Zendesk skill and CLI based on the skill-project-management setup pattern.

## Scope
Add source-checkout setup scripts, installed skill copy, local bin install, Claude/Codex skill links, install-state metadata, safe reinstall flow, and release config for the supported platform matrix.

## Acceptance Criteria
The repo includes working setup scripts and release config for macOS arm64, macOS x86_64, Windows x86_64, and Windows arm64.
